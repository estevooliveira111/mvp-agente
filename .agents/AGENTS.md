# Project Guidelines

## Naming Conventions
- **Language**: O código deve seguir integralmente o padrão em inglês.
- **Identifiers**: Todos os nomes de variáveis, funções, classes, arquivos, diretórios, endpoints e demais identificadores devem estar em inglês.
- **Natural Language**: Apenas os prompts, mensagens exibidas ao usuário e comentários no código devem permanecer em português.
- Mantenha esse padrão de forma consistente em todo o projeto.
